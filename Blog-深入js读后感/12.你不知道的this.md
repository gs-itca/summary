<h1 align=center>你不知道的this</h1>

> this关键字是JavaScript中最复杂的机制，在缺乏清晰的认识的情况下，this对你来说完全是一种魔法。

## 一、为什么要用this，理解它对你有什么帮助？

先来看下面的代码

```js
function identify() {
    return this.name.toUpperCase();
}
function speak() {
    var greeting = "Hello, I'm " + identify.call( this );
    console.log( greeting );
}
var me = {
    name: "Kyle"
};
var you = {
    name: "Reader"
};
identify.call( me ); // KYLE
identify.call( you ); // READER
speak.call( me ); // Hello, 我是 KYLE
speak.call( you ); // Hello, 我是 READER
```

你自己推断的结果，和真是结果对应上了吗？说实话，在实际开发中不会有那么多绕来绕去的this绑定来增加代码的复杂性，但是它出现时，你必须要读懂～
