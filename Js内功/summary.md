# 深入系列

**这一系列主要的是从另一个角度去学习JS的基础部分**

刚参加工作的时候，一直就纠结于怎么解决某一个bug，只要“看不见”这个bug，就当时解决了问题，将它抛在脑后。但很快意识到这是不对的，借用最近看到的一句话“我不是什么都会，我只不过能将我会的都研究的比较明白”。

就好比你学会了一道题 a + b = c,你知道了 a与b可以得到c，但a是什么？，a又是谁得到的？凡事向深层多思考一下总是没错的。我们一起加油，死磕js～
